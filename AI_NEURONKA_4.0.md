### 4. ✅ AI понимает контекст

**История диалога:**
```python
messages=[
    {"role": "user", "content": "хочу маршрут"},
    {"role": "assistant", "content": "С кем планируете?"},
    {"role": "user", "content": "с другом"},
    {"role": "assistant", "content": "Сколько времени?"},
    {"role": "user", "content": "30 минут"},
]
```

**AI анализирует:**
- Количество сообщений (`len(messages)`)
- Извлечённые данные (`profile`)
- Текущий маршрут (`route`)

**Решение:**
- 0-1 ответов → задать вопросы
- 2-3 ответа → генерировать маршрут
- "не хочу" → заменить место
