## 🚀 Установка

### 1. Клонирование репозитория
```
git clone https://github.com/tasbafra/
```
### 2. Установка зависимостей Python
```
pip install -r requirements.txt
```

### 3. Запуск приложения

```
bash ./run.sh
```
```
shell ./run.bat
```
