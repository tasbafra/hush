# ✅ УЛУЧШЕНИЯ ЧАТА — ВЕРСИЯ 3.2

## 🎯 ЧТО УЛУЧШЕНО

### 1. ✅ Увеличены токены Groq до 1000
**Файл:** `backend/groq_service.py`

**Было:**
```python
max_tokens=350  # decision
max_tokens=200  # reply
```

**Стало:**
```python
max_tokens=1000  # decision ✅
max_tokens=1000  # reply ✅
```

**Результат:**
- Более развёрнутые ответы от AI
- Может предложить больше деталей маршрута
- Лучше понимает контекст

---

### 2. ✅ Подсказки о возможности замены мест
**Файлы:** `backend/groq_service.py`, `Web/src/AssistantChat.tsx`

**Где отображаются:**

#### A. В сообщении AI
```
AI: Вот ваш маршрут: 1. Площадь Ленина 2. Набережная...
    Собрать этот маршрут?
    
💡 Не нравится место? Нажмите ✕ рядом с названием чтобы заменить.
```

#### B. После подтверждения маршрута
```
✅ Маршрут сохранён! Перехожу к деталям...

💡 Подсказка: Нажмите на ✕ рядом с местом чтобы заменить его на другое.
```

#### C. Визуальные кнопки для каждого места
```
💡 Нажмите на место чтобы заменить:
[✕ Площадь Ленина] [✕ Набережная] [✕ Парк культуры]
```

**Стили кнопок:**
- Красная обводка (rgba(239,68,68,0.5))
- Hover эффект (поднимается на 1px)
- Обрезают длинные названия (>25 символов)
- Показывают до 5 мест

### 3. ✅ AI сам предлагает замену
**Файл:** `backend/groq_service.py`

**Системный промпт обновлён:**
```
"- В ответе упомяни что места можно заменить 
   (напиши 'места можно заменить' или 'можно заменить')"
```

**Пример ответа AI:**
```
AI: Для быстрой прогулки рекомендую: Площадь Ленина → 
    Набережная → Парк культуры. Места можно заменить 
    если что-то не понравится!
    
    Собрать этот маршрут?
```
