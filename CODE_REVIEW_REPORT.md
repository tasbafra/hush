## ⚠️ Проблемы и замечания по коду

### Критичные (нужно исправить до демо)

1. **Мало мест в базе** (`places.json`)
   - Сейчас: 5 мест
   - Нужно по ТЗ: 20-25 мест
   - **Кто:** Алина
   - **Приоритет:** 🔴 Высокий

2. **Нет мест с `activity_level="high"`**
   - В текущей базе все места имеют `activity_level="low"`
   - Логика `tired_mode` для удаления активных мест не тестируется на реальных данных
   - **Кто:** Алина (добавить места), Сева (протестировать)

### Средней важности

3. **Дублирование функции `get_place_by_id`**
   - Есть в `main.py` и в `replanner.py`
   - **Решение:** Оставить только в `replanner.py`, импортировать в `main.py`

4. **Отсутствие валидации входных данных**
   - Нет проверки на `None` в `all_places`
   - Нет проверки на пустые `current_stop_ids` (частично есть)

5. **Жёсткое кодирование лимита 5 мест**
   - В `_handle_coffee_nearby()` лимит "5" захардкожен
   - **Решение:** Вынести в константу `MAX_ROUTE_LENGTH`

### Низкой важности (косметика)

6. **Отсутствуют type hints для внутренних функций**
   - Частично есть, но не везде

7. **Отсутствует логирование**
   - Полезно для отладки на сервере
